# Return an empty numeric if all elements are `NA`.

Return an empty numeric if all elements are `NA`.

## Usage

``` r
empty_vector_if_na(x)
```

## Arguments

- x:

  (`numeric`)  
  vector.

## Value

An empty `numeric` if all elements of `x` are `NA`, otherwise `x`.

## Examples

``` r
x <- c(NA, NA, NA)
# Internal function - empty_vector_if_na
```
