# Description of method used for proportion comparison

**\[stable\]**

This is an auxiliary function that describes the analysis in
[`s_proportion_diff()`](https://insightsengineering.github.io/tern/reference/prop_diff.md).

## Usage

``` r
d_proportion_diff(conf_level, method, long = FALSE)
```

## Arguments

- conf_level:

  (`proportion`)  
  confidence level of the interval.

- method:

  (`string`)  
  the method used for the confidence interval estimation.

- long:

  (`flag`)  
  whether a long (`TRUE`) or a short (`FALSE`, default) description is
  required.

## Value

A `string` describing the analysis.
