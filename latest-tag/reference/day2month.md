# Conversion of days to months

Conversion of days to months

## Usage

``` r
day2month(x)
```

## Arguments

- x:

  (`numeric(1)`)  
  time in days.

## Value

A `numeric` vector with the time in months.

## Examples

``` r
x <- c(403, 248, 30, 86)
day2month(x)
#> [1] 13.2402464  8.1478439  0.9856263  2.8254620
```
