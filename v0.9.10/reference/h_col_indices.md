# Obtain column indices

**\[stable\]**

Helper function to extract column indices from a `VTableTree` for a
given vector of column names.

## Usage

``` r
h_col_indices(table_tree, col_names)
```

## Arguments

- table_tree:

  (`VTableTree`)  
  `rtables` table object to extract the indices from.

- col_names:

  (`character`)  
  vector of column names.

## Value

A vector of column indices.
