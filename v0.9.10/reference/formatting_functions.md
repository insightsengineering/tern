# Formatting functions

See below for the list of formatting functions created in `tern` to work
with `rtables`.

## Details

Other available formats can be listed via
[`formatters::list_valid_format_labels()`](https://insightsengineering.github.io/formatters/latest-tag/reference/list_formats.html).
Additional custom formats can be created via the
[`formatters::sprintf_format()`](https://insightsengineering.github.io/formatters/latest-tag/reference/sprintf_format.html)
function.
